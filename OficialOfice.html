Добро пожаловать в Word Clone

Это упрощенный клон Microsoft Word, созданный с использованием HTML, CSS и JavaScript.

Вы можете редактировать этот текст, используя панель инструментов выше. Попробуйте изменить шрифт, размер текста или применить стили.

Для форматирования выделите текст и нажмите на соответствующие кнопки на панели инструментов.

  • Создавайте документы
  • Форматируйте текст
  • Сохраняйте и печатайте

Word Clone - это упрощенный текстовый редактор с основными функциями:

  • Форматирование текста (жирный, курсив, подчеркивание)
  • Изменение шрифта и размера текста
  • Выравнивание текста
  • Создание списков
  • Применение стилей заголовков
  • Подсчет слов
  • Масштабирование документа

Для форматирования выделите текст и используйте кнопки на панели инструментов.

Word Clone v1.0

Упрощенный клон Microsoft Word, созданный на чистом HTML, CSS и JavaScript.

Данное приложение демонстрирует возможности создания офисных приложений в браузере.

Функции реализованы с использованием ContentEditable API и команд document.execCommand.
